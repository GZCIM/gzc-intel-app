Portfolio Component Test

Open the browser console and run these commands to test:

// 1. Check if portfolio exists in inventory
componentInventory.getComponent('portfolio')

// 2. Check all components
componentInventory.searchComponents('').map(c => ({ id: c.id, name: c.displayName }))

// 3. Check financial category
componentInventory.getComponentsByCategory('financial')

// 4. Search for portfolio
componentInventory.searchComponents('portfolio')

// 5. Force re-add portfolio if needed
if (!componentInventory.getComponent('portfolio')) {
  componentInventory.addComponent({
    id: 'portfolio',
    name: 'Portfolio',
    displayName: 'Portfolio Dashboard',
    category: 'financial',
    subcategory: 'portfolio',
    description: 'Comprehensive portfolio management with real-time P&L, positions table, metrics, and charts',
    defaultSize: { w: 8, h: 6 },
    minSize: { w: 6, h: 4 },
    maxSize: { w: 12, h: 10 },
    tags: ['portfolio', 'trading', 'pnl', 'positions', 'financial', 'real-time', 'dashboard'],
    complexity: 'complex',
    quality: 'enhanced',
    source: 'internal'
  })
  componentInventory.rebuildSearchIndex()
  console.log('Portfolio added!')
}

Step-by-Step Guide:

  1. Create Dynamic Tab:
    • Click the + button in header
    • Name it "Portfolio Test"
    • Select "Dynamic" type
  2. Enter Edit Mode:
    • Click "Edit" button (top-right of tab)
    • Button should change to "Save"
  3. Add Component:
    • Click "âž• Add Component" button
    • Modal should open with component list
  4. Find Portfolio:
    • Look for "Portfolio Dashboard"
    • Should be in the "Local Components" tab
    • Category: financial
  5. Select and Position:
    • Click on Portfolio Dashboard
    • It will appear in the canvas
    • Drag to position
    • Resize by dragging edges

Troubleshooting:

If Portfolio doesn't appear in the list:

  1. Check browser console for errors
  2. Run the force re-add command above
  3. Refresh the page
  4. Check the InventoryDebugger (bottom-right)
